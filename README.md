# LUXECart Ecommerce Landing Page

LUXECart is a modern ecommerce landing page built with HTML, CSS, and JavaScript. The project features a premium glassmorphism inspired design with responsive layouts, smooth animations, and interactive shopping functionality.

Key features include:

* Responsive navigation with mobile menu support
* Modern hero section with animated visual elements
* Product categories and featured product displays
* Interactive shopping cart with dynamic updates
* Toast notifications for user actions
* Newsletter subscription form with email validation
* Customer testimonials and promotional sections
* Fully responsive design optimized for desktop, tablet, and mobile devices

This project demonstrates modern frontend development practices while creating an elegant and engaging shopping experience. It is ideal for learning responsive web design, DOM manipulation, and interactive user interface development using vanilla JavaScript.
